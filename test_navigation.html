Тест навигации AI-NK

Инструкции:

1. Откройте https://localhost

2. Проверьте левое навигационное меню

3. Найдите пункт "Нормоконтроль - 2"

4. Нажмите на него для перехода

Ожидаемые элементы в навигации:

  • ✅ Дашборд
  • ✅ Чат с ИИ
  • ✅ Выходной контроль корреспонденции
  • ✅ Консультация НТД от ИИ
  • ✅ Объекты аналоги
  • ✅ Нормоконтроль
  • ✅ Нормоконтроль - 2 (новый модуль)
  • ✅ Нормативные документы

Если модуль не виден:

  1. Обновите страницу (Ctrl+F5 или Cmd+Shift+R)
  2. Очистите кэш браузера
  3. Проверьте консоль браузера на ошибки (F12)
  4. Попробуйте в режиме инкогнито

Проверка API:

API модуля доступен по адресу: http://localhost:8010/docs

Health check: http://localhost:8010/health
